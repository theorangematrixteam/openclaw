# Samruddhi Task Management

## Overview
This folder tracks all tasks assigned to Samruddhi, a content creator at Orange Matrix.

## Structure

### `tasks.json`
Master task database with all assignments, status, priorities, and deadlines.

### `daily-assignments/`
Daily task lists created each morning for Samruddhi.

### `performance-log.md`
Weekly performance notes and feedback.

## Workflow

1. **Morning (9-10 AM):** Review pending tasks, assign daily priorities
2. **End of Day:** Samruddhi reports completion status
3. **Weekly:** Review completed work, provide feedback, plan next week

## Task States
- `pending` — Not started
- `in-progress` — Currently working
- `review` — Completed, awaiting review
- `done` — Approved and complete
- `blocked` — Cannot proceed, needs input

## Priority Levels
- `P0` — Must finish today
- `P1` — Finish today if possible
- `P2` — This week
- `P3` — Backlog
