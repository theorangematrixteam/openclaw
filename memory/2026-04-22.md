## Totsburg 2-Kid Photo Generation

### Workflow

2. Upscaled via SeedVR2 to 4096px
3. SAM3 crop mask on clothing (tshirt and shorts)

### Iterations

### Key Rule Learned
**Crop workflow = final output for editing. Stitch = only when explicitly asked.**
Jinay had to repeat this. DON'T stitch back unless told to.

### Per-Kid Mask Splitting
- Problem: SAM3 outputs individual masks but InpaintCropImproved combines them into one crop
- Fix: Created `run_crop_per_mask.py` — 2-pass approach:
  1. Run SAM3 to get combined mask
  2. Split into individual masks using scipy connected components
  3. Run InpaintCropImproved separately for each individual mask
- Uses ImageToMask node to convert uploaded mask image back to MASK type for InpaintCropImproved
- Output naming: `crop_kid1_basename.png`, `crop_kid2_basename.png`, etc.
