# Research Order

## Purpose

This planner should not behave like a random idea generator.
It should behave like a strategist.

## Required order

### 1. Client understanding

Read the client knowledge file first.

Capture:

- category
- offer
- positioning
- ICP
- content direction
- messaging constraints
- what not to do

### 2. Competitor / reference review

Study provided competitors or reference brands.

Look for:

- recurring content patterns
- positioning language
- visual style
- overused hooks
- weak spots
- whitespace / opportunity

Do not imitate them blindly.

### 3. Calendar relevance

Check relevant:

- special days
- seasonal windows
- launch moments
- category timing

Only keep what is strategically relevant.

### 4. Plan construction

Now build the actual monthly plan.

Each row should have:

- reason
- audience relevance
- brand fit
- creation usefulness

## Reason framework

A post usually belongs in the plan for one of these reasons:

- build awareness
- create desire
- educate
- reduce objections
- create urgency
- support conversion
- support brand positioning

If no reason is clear, drop the post.

## Brand-specific caution

### Proofit

Lean into:

- evidence
- hidden issues
- decision confidence
- inspection proof
- workmanship / risk / damage / prevention

### Notch India

Lean into:

- stylish but wearable fashion
- age-inclusive appeal
- launch energy
- product showcase with taste
- brand mood and identity building
