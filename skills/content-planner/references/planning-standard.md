# Planning Standard

## Baseline

Use `Proofit` as the stronger benchmark for monthly planning quality.

Compared with `Notch India`, the Proofit plan is stronger because it includes:

- clearer topic framing
- stronger concept/objective writing
- better post angle definition
- CTA per row

## Minimum viable row quality

Each row should be specific enough that a creator can start work from it.

A row is weak if it only says something like:
- new reel
- product shot
- something aesthetic

A row is stronger when it includes:
- hook
- objective
- angle
- CTA

## Recommended planning logic

For each month:

1. Decide posting frequency
2. Assign content buckets across the month
3. Add special-day opportunities only where relevant
4. Define each post with enough detail to reduce back-and-forth later

## Good bucket balance

A healthy monthly plan usually balances:

- authority/proof
- education
- product/offer visibility
- conversion intent
- timely relevance

Exact mix depends on the client.

## Client-facing clarity

Remember the client sees this sheet.

So the plan should feel:
- smart
- clear
- organized
- easy to approve

It should not feel like internal brainstorming mess.
